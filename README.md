# 李洛維 Lo-Wei Lee, Ph.D.

**智慧政府・資料治理・AI 治理** ｜ 臺灣警察專科學校 助理教授

Assistant Professor at Taiwan Police College. Ph.D. in Public Administration, National Chengchi University. I study **smart government, data governance, and algorithmic accountability** — how the state adopts AI, and how institutions keep it answerable.

Before academia, I spent **12 years as a civil servant**, so my research agenda starts from how government actually works, not just how theories say it should.

**Research toolkit**: computational text analysis (NLP, structural topic models), survey experiments, SEM, and mixed-methods institutional analysis.
